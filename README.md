# Online-Campaign-and-eCommerce-Analysis
Analyzed Google and Facebook ad performance using PostgreSQL and BigQuery. Calculated ROMI, reach growth, and campaign value across platforms. Extracted GA4 eCommerce funnel data to measure session-to-purchase conversions by traffic source, preparing structured datasets for BI reporting.

Tools: SQL, PostgreSQL (DBeaver), Google BigQuery, Google
Analytics 4

Overview: Analyzed online advertising performance and
eCommerce metrics using real GA4 data to evaluate marketing
effectiveness.

Responsibilities: Queried and aggregated ad data, calculated
ROMI and conversion rates, and prepared datasets for BI
dashboards.

Result: Delivered clean, structured data and insights that
improved understanding of campaign ROI and user behaviour.
